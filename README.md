# BookMind AI - دليل المستخدم والمنصة | User Guide & Platform Overview

منصة **BookMind AI** هي بوابة متكاملة لإدارة المعرفة الدلالية ومحرك بحث ذكي يعتمد على تقنيات الـ **RAG (Retrieval-Augmented Generation)** لاستخراج المعرفة الذكية من الكتب والوثائق والمستندات التقنية. تتميز المنصة بواجهة مستخدم غنية وعالية الجودة مستوحاة من الطراز المكتبي العريق بالألوان المتناغمة (الأخضر الداكن والذهبي الفاخر مع خلفية عاجية مريحة للعين).

## 🚀 الميزات الأساسية | Core Features

### 1. الإدخال والاستخلاص الذكي | Smart Document Ingestion
*   **دعم سحب وإفلات الملفات:** يدعم صيغ الملفات الأكثر انتشاراً مثل `PDF` و `TXT` و `DOCX` بالإضافة إلى إمكانية لصق النصوص الطويلة مباشرة.
*   **محلل نصوص متطور (Text Parser):** تم تحسين الخادم للتكامل السلس والكامل مع مكتبة `pdf-parse` الحديثة لمعالجة وقراءة ملفات الـ PDF الكبيرة واستخلاص كافة الحروف والصفحات والبيانات الوصفية بنجاح.
*   **خطوات معالجة مرئية (Stepper):** تتبع تقدم الملف من مرحلة الرفع، التحليل الهيكلي، التقطيع الدلالي، التوجيه المتجهي، وحتى الإدراج النهائي في قاعدة المعرفة الدلالية.

### 2. الفهرسة والتقطيع الدلالي | Semantic Chunking & Vectorization
*   **خوارزمية bge-m3 المتطورة:** تقطيع الملفات بطريقة دلالية ذكية وتحويلها إلى متجهات ذات 1024 بعداً (1024-D Vectors).
*   **التحكم الدلالي المتقدم (Similarity Sandbox):** شريط تمرير تفاعلي لتحديد حساسية الحدود الدلالية وعتبة التشابه (Similarity Threshold) لتوليد فواصل دقيقة ومتجانسة للمحتوى.
*   **مستعرض القطاعات الدلالية (Semantic Viewer):** استعراض حي لكافة القطاعات المولدة ورسم توضيحي مصغر للمتجهات الخاصة بكل جزء.

### 3. مجموعات المعرفة المخصصة | Knowledge Collections
*   تصنيف الكتب والمقالات في مجموعات موضوعية ملونة (مثل: الفلسفة والمنطق، التكنولوجيا والذكاء، الروايات والأدب، علم الاجتماع والتاريخ).
*   إمكانية إنشاء مجموعات مخصصة باسم ووصف وأيقونة دلالية مميزة لتخصيص سياق الـ RAG.

### 4. المحادثة الذكية المعتمدة على الـ RAG | Interactive RAG Chat
*   توجيه الأسئلة والاستعلام بمرونة كاملة عبر كامل المكتبة أو حصر المحادثة في كتاب أو مستند تقني محدد.
*   **نظام الاستشهادات الموثوق (Source Citations):** يظهر الروبوت المراجع والصفحات المقتبس منها مع عرض **نسبة المطابقة الدلالية** وفقرة الاقتباس الحرفية المظللة بالذهبي والأخضر.
*   تحديثات حية تعكس حالة التفكير والمحاكاة (Retrieving ➜ Ranking ➜ Generating).

### 5. الرمل المتجهي (Vector Search Sandbox)
*   لوحة تجريبية تفاعلية للمطورين والباحثين لاختبار مطابقة الكلمات واسترجاع الأجزاء المتشابهة مباشرة ومقارنة متجهات الأوزان في الوقت الفعلي.

### 6. دعم نماذج التوليد المحلية والمجانية بالكامل (Ollama Integration)
*   **تشغيل محلي بالكامل (Private & Offline):** إمكانية توجيه الأسئلة واستنباط الإجابات بالاعتماد على خادم Ollama محلي بدلاً من خطوط الاتصال السحابية.
*   **إدارة مخصصة من الإعدادات:** لوحة متكاملة لتهيئة مسار الاتصال (Endpoint Socket)، اختيار اسم المعرف للنموذج (مثال: `qwen2.5:14b` أو `llama3`)، واختبار الاتصال فورياً بضغطة زر مع جلب قائمة النماذج النشطة.
*   **نظام التبديل المتعدد:** دعم لكل من الاتصال كـ "وسيط خادم (Server Proxy)" لتأمين مكالمات API، أو "اتصال مباشر من المتصفح (Browser Direct)".

## 🛠️ دليل التشغيل والتثبيت للمطورين | Setup Instructions

### المتطلبات الأساسية
*   بيئة تشغيل **Node.js** (إصدار 18 فما فوق).
*   الحصول على مفتاح واجهة برمجة تطبيقات Gemini ومثبت في متغيرات البيئة باسم:
    ```env
    GEMINI_API_KEY=your_gemini_api_key_here
    ```

### تشغيل المنصة في بيئة التطوير
1. قم بتثبيت الاعتماديات المطلوبة:
   ```bash
   npm install
   ```
2. ابدأ خادم التطوير المدمج (Express + Vite) والذي يعمل كمنصة كاملة (Full-stack) على المنفذ `3000`:
   ```bash
   npm run dev
   ```
3. لبناء التطوير للإنتاج والتشغيل المستقر:
   ```bash
   npm run build
   npm start
   ```

### 🧠 تشغيل نموذج Ollama المحلي والمجاني (Offline / Local Mode Setup)

للاستفادة من توليد الإجابات بشكل مجاني، محلي بالكامل وبدون الاتصال بالإنترنت:

1. **تحميل وتثبيت Ollama:**
   قم بتحميل بيئة التشغيل من الموقع الرسمي [ollama.com](https://ollama.com) وتثبيتها على حاسوبك الشخصي.
2. **سحب النموذج المناسب (Pull Model):**
   شغّل سطر الأوامر (Terminal) واكتب الأمر التالي لتحميل النموذج المفضل لديك (مثال: نموذج `qwen2.5:14b` أو `llama3` أو `mistral`):
   ```bash
   ollama pull qwen2.5:14b
   ```
3. **تفعيل إعدادات CORS وسماح الوصول (هام جداً):**
   لكي يسمح المتصفح أو الخادم لـ BookMind AI بتبادل البيانات والاتصال المباشر بـ Ollama، يجب تشغيل الخدمة مع تمكين الأصول المشتركة عبر تعيين متغير البيئة `OLLAMA_ORIGINS`:
   * **على أنظمة Linux/macOS:**
     ```bash
     OLLAMA_ORIGINS="*" ollama serve
     ```
   * **على نظام Windows (عبر PowerShell):**
     ```powershell
     $env:OLLAMA_ORIGINS="*"
     ollama serve
     ```
4. **تفعيل وتثبيت الإعدادات في واجهة BookMind AI:**
   * انتقل إلى **شريط الإعدادات الفنية (Settings)** في التطبيق.
   * تحت قسم "محرك التوليد ونماذج الذكاء الاصطناعي"، اختر نموذج **Ollama (Local & Offline LLM)**.
   * اضبط الرابط (الافتراضي هو `http://localhost:11434`) واسم النموذج (مثل `qwen2.5:14b`).
   * انقر فوق **اختبار الاتصال بالنموذج المحلي (Test Ollama Connection)** للتحقق من سلامة الاتصال وجلب النماذج المثبتة بنجاح على جهازك.

## 📖 إرشادات الاستخدام اليومي | How to Use BookMind AI

1.  **لوحة التحكم الرئيسية:** تقدم لمحة إحصائية وافية لعدد الكتب والمجموعات والمقاطع المخزنة، بالإضافة إلى مخطط لتوزع لغات الكتب، ولوحة حية لمراقبة استهلاك عتاد خادم الـ RAG والمخزن المؤقت لـ ChromaDB.
2.  **المكتبة وإضافة المستندات:** انتقل لتبويب المكتبة لتصفح كتبك. انقر فوق زر "إضافة كتاب جديد" لفتح واجهة الرفع السلسة، ارفع ملف PDF الخاص بك أو الصق النص، اختر اللغة والتصنيف والمجموعة، ثم انقر فوق "إدخال الملف لقاعدة المعرفة".
3.  **المستكشف الدلالي:** استكشف هيكلة ملفك الداخلية، واضبط عتبة التشابه لمشاهدة تأثيرها على تقسيم النصوص المتجهية في التو واللحظة.
4.  **المحادثة المتقدمة:** اطرح الأسئلة الصعبة على الروبوت، وشاهد كيف يقدم إجابة مسبوكة مستندة بالكامل على البيانات الحرفية الموثقة للملفات مع تفصيل دقيق لصفحات وهوامش الاقتباس.
